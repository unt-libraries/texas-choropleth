Clarification of Terms

Scale

The scale choice is crucial in determining how effective the choropleth is. Based on the nature of your dataset, you will want to choose one of the following to make the map more meaningful.

Quantized

This is a variant of a linear scale. Instead of a continuous domain, like a linear scale, the input domain is divided equally into the number of specified classes. Eventually, each class is differentiated by a specific color in the choropleth. This scale is available for any dataset.

Logarithmic

A nonlinear scale based on the orders of magnitude unlike a standard linear scale. This scale can be useful when dataset values grow exponentially. Because the logarithmic scale output approaches inifinity as the input approaches 0, this scale is only available to datasets where the values are exclusively negative, or exclusively positive.

Scheme

The schemes used in this app, are the same schemes that Cynthia Brewer used to develop and explain the ColorBrewer palettes, which are also used in this app.

Palette

A palette, in the context of this application, is a collection of colors that was intentially assembled to communicate the scheme. This application uses the palettes that Cynthia Brewer has created specifically designed for maps. See ColorBrewer for more information.

Classes

For choropleths that employ the Quantized scale, the number of classes determines the number of groups that your dataset is broken into, thus how many colors will be represented. The number of classes available is determined by the palette. Each palette includes at least 3 and as many as 12 classes.
