Purpose

This project was created to allow users to explore and create data-driven visualizations of the state of Texas. Choropleth maps are a popular and flexible model for representing data for a geographic region. This project lowers the techonology barrier to allow average users to simply upload a dataset, and easily configure how that data should be represented within a choropleth map.

What is a choropleth map?

A choropleth map is a thematic map in which areas are shaded or patterned in proportion to the measurement of the statistical variable being displayed on the map, such as population density or per-capita income.
